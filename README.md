# Guía de Clonación del Repositorio

## Requisitos Previos

Tener instalado Git. Para verificar:
```bash
git --version
```

Si no lo tienes, descárgalo desde [git-scm.com](https://git-scm.com/)

## Clonar el Repositorio

### 1. Abrir la terminal

- Windows: Git Bash, CMD o PowerShell
- Mac/Linux: Terminal

### 2. Navegar a la carpeta deseada

```bash
cd ruta/donde/quieres/clonar
```

### 3. Ejecutar el comando de clonación

```bash
git clone https://github.com/JCbral04/ING-WEB-A3.git
```

### 4. Ingresar al directorio

```bash
cd ING-WEB-A3
```

## Verificar la Clonación

```bash
git status
```

## Actualizar el Repositorio

Para obtener los últimos cambios:

```bash
git pull origin main
```

## Comandos Básicos

Ver estado:
```bash
git status
```

Ver historial:
```bash
git log --oneline
```

Ver ramas:
```bash
git branch -a
```
